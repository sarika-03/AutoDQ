# dq-pilot
dq-pilot is an AI-powered data quality recommender built on OpenMetadata. 
Point it at any table and it will analyze column types, null rates, value 
distributions, and descriptions to suggest relevant test cases from 
OpenMetadata's test library complete with parameters and plain-English 
reasoning for human review.s
